Design principles — distilled from the Claude Code reference

  1. True neutrals, no cast. Dark mode sits at #0E0E0E-#1A1A1A (near-black, no blue tint). Light mode is #FAFAF9-#F2F1EE paper (warmth ≤ Δ3). Replaces 812156c's warmer tri-surface workspace stack.
  2. One accent, used sparingly. Periwinkle #7AA5FA / #4C81DD — active deck row, primary CTA, chart data. No status-color decoration.
  3. Chrome subtraction. Borders are 0.5pt tonal shifts (#181818 / #E7E5E1), never visible strokes. Shadows only on the outer shell. Hover = 4–5% tint, no lift.
  4. Typography leads. Section labels 10px UPPERCASE @ 42% opacity + 0.9px tracking. Row names 12.5px / 400 idle → 500 active. Hierarchy via weight/tone, not dividers.
  5. Deck-color restraint. 8-hue palette from d0ababb kept but desaturated — S≈0.38, L≈0.58. Mono letter on the swatch so the deck name leads, not the color.
  6. Flush chat drawer. Sits on the canvas like a sheet. Top-border + 18px backdrop-blur; no floating shadow. Composer input is the bright piece, drawer body recedes.
  7. Inspector is a list, not a panel. Right column is key/value rows with bottom-border ticks, plus a three-action stack. No card-in-card nesting.
  8. Canvas breathes. 22px grid padding, 16px gaps, cards are pure 16:9 with a gradient scrim + footer overlay — no title row under the card (footer moves inside per 812156c, which we keep).
